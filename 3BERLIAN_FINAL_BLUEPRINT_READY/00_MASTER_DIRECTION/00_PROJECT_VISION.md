# PROJECT VISION

## Objective
Menetapkan visi jangka panjang aplikasi 3 Berlian agar seluruh refactor tetap searah.

## Product Direction
Aplikasi 3 Berlian diarahkan dari aplikasi rental pakaian adat menjadi:

**Sanggar Management Super App**

Yang kelak dapat menangani:
- rental pakaian adat
- aksesoris dan properti
- jasa tari
- MC
- penyanyi
- musik tradisional
- paket acara
- jadwal tampil
- booking event
- pembayaran DP dan pelunasan
- laporan operasional
- talent management

## Current Phase
Fase sekarang **bukan** implementasi seluruh fitur jasa.

Fase sekarang adalah:
- mematangkan UI/UX
- menstabilkan transaksi
- merapikan mobile density
- menyiapkan arsitektur agar nanti siap multi-service

## Core Principles
1. **Mobile-first**  
   Aplikasi harus nyaman dipakai di HP seperti aplikasi kasir/superapp.

2. **Operational speed**  
   Kasir/operator harus bisa bekerja cepat tanpa banyak scroll.

3. **Compact elegant UI**  
   UI harus padat, tetapi tetap premium dan tidak terlihat murahan.

4. **Modular expansion**  
   Fitur baru seperti MC, tari, event, dan paket acara harus bisa masuk tanpa redesign total.

5. **Safe refactor**  
   UI boleh dirapikan, tetapi alur transaksi, stok, dan laporan tidak boleh rusak.

## Non-Goals Saat Ini
- Jangan implementasi penuh booking tari/MC dulu.
- Jangan rename besar-besaran collection Firebase.
- Jangan rewrite seluruh aplikasi.
- Jangan menyentuh login/account dulu.
