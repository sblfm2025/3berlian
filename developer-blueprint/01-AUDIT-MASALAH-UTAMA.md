# Audit Masalah Utama

Masalah kritis:
1. Checkout dan stok belum cukup aman jika terjadi transaksi bersamaan.
2. Struktur stok perlu distandarkan.
3. Transaksi tidak boleh dihapus permanen.
4. Validasi checkout harus dipusatkan.
5. UI masih perlu menghindari alert browser.
6. Return flow perlu mendukung kondisi barang.
7. Laporan harus membaca data transaksi secara konsisten.
8. Mobile POS harus lebih cepat digunakan.
