# QA Checklist

Checkout:
- cart kosong ditolak
- stok habis ditolak
- invoice terbentuk
- stok available turun
- stok rented naik
- audit log tercatat

Return:
- good kembali ke available
- laundry masuk laundry
- damaged masuk damaged
- partial return benar
- full return benar

Void:
- wajib alasan
- stok kembali jika aktif
- transaksi tetap ada

Reports:
- void tidak masuk omzet
- denda benar
- deposit benar

Mobile:
- cart sticky
- tombol besar
- tidak ada tabel melebar
